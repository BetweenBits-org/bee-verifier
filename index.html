bee-verifier · 독립 정산 영수증 검증기

Independent settlement-receipt verifier — 서버 없이 Bitcoin 체인 + 수학으로만 검증
🔓 완전 독립 검증 — 이 페이지는 공개 소스(bee-verifier)입니다. 검증 과정에서 BitCert 서버에 접속하지 않습니다. 오직 아래에 지정한 공개 Bitcoin 익스플로러와 브라우저 안의 계산(SHA-256)만 사용합니다. 필요한 입력은 영수증 PDF와 인스크립션 ID 둘뿐입니다.

① 검증 입력

📄 영수증 PDF를 여기에 끌어다 놓거나 클릭하여 선택
고급 — 오프라인 모드 (익스플로러 없이 raw tx hex로 검증)
오프라인 모드는 raw reveal hex로 witness↔영수증 해시 대조만 수행합니다. 인스크립션 ID를 입력하면 공개 익스플로러에서 자동 조회합니다.

검증 결과

bee-verifier — Project Bee S4 정산 영수증(Taproot commit/reveal · "beecert" witness 인스크립션) 독립 검증기.
